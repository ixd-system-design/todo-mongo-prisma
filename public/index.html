TODO list app created by Harold Sikkema in the context of Systems Design (Databases and Dynamic Data) in the Interaction Design Program at Sheridan College. This app uses MongoDB Atlas to persist a collection of TODO items. It communicates with MongoDB using a Mongoose Schema via a REST API built with Express. API endpoints illustrate a mapping of CRUD operations onto HTTP Verbs. Data storage is handled on the frontend via a set of functions: createData, readData, updateData, deleteData, as per the the acronym CRUD. To keep things modular, functions are imported from a separate module, crud.js
